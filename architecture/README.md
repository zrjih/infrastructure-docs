# System Architecture

## Overview

The 2BINTEC infrastructure runs on a single OVH VPS (`141.95.159.128`) with Docker-based containerization, network segmentation, and reverse proxy with SSL termination.

## Data Flow

1. **Client** вҶ’ DNS resolves to `141.95.159.128`
2. **nginx_proxy** (port 80/443) terminates SSL, routes by `server_name`
3. HTTP вҶ’ 301 redirect to HTTPS
4. HTTPS вҶ’ proxy_pass to internal container (by Docker DNS name)
5. Frontend serves static React build
6. Frontend calls `/api/` вҶ’ nginx proxies to backend container
7. Backend вҶ’ MongoDB (internal Docker network, no public port)
